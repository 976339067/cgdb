Documentation

When CGDB was originally created, it was just to get a fast, responsive GDB front end to work on some very slow SunOS's. Originally, we never intended for CGDB to become the replacement front end to GDB on every machine we used. However, as CGDB developed, it was clear that no other front end could help us get our development tasks done in as fast a time. Since we find CGDB to be so usefull, an attempt is being made to keep as much information possible, organized in one place, so that CGDB user's can find answers to all there questions. Below is some of the documentation that we have been beggining to gather. Enjoy!
