Anonymous CVS

The most recent changes to CGDB can be found by checking out a copy of the current CVS tree. You may want to try this to see if a bug is resolved before reporting the error. It should be noted that the CVS tree is not guaranteed to be stable and is not tested as well as the latest tarball probably is. To get a copy of the CVS tree, run the following commands:

cvs -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/cgdb login
(blank password)

cvs -z3 -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/cgdb co cgdb

If you don't have CVS, visit their home page.
